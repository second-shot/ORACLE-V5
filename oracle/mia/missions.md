# Mia Missions

## Mission Format
Each mission must include:
- name
- purpose
- priority
- status
- owner
- output
- definition of done

---

## MIA-001 — Map Oracle
- Priority: P1
- Status: ready
- Owner: Mia
- Purpose: create a clear snapshot of Oracle's current architecture and readiness
- Output: `oracle/docs/architecture.md`
- Definition of done:
  - current structure reviewed
  - existing docs listed
  - missing elements identified
  - system boundaries described

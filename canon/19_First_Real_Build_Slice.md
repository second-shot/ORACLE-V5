# First Real Build Slice

## Purpose
Define the first version that proves Oracle is real.

## Must prove
- raw input can be captured
- raw input becomes structured objects
- objects can be classified and scored
- objects can be routed correctly
- the archive feels intelligent
- at least one finished output can be produced
- pricing can return a route, not just a number

## Build first
1. login / onboarding
2. command surface
3. object model
4. compression + classification + score + route
5. living home
6. archive skeleton
7. one pricing route output
8. one proposal or listing output

## Do not build yet
- over-deep automation
- unnecessary agent swarm logic
- decorative complexity
- too many account types
- too many integrations
- fake intelligence layers

## Success condition
A user can enter raw material, see it become a structured object, watch it route into the right path, and receive one usable output.
